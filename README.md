# SmartApply AI

SmartApply AI is an AI-powered CV optimization platform that helps job seekers improve their applications by comparing their CVs to job descriptions.

## Features
- CV and job description matching
- Match score analysis
- Missing skills detection
- AI-generated CV improvements
- AI-generated cover letters

## Tech Stack
- ASP.NET Core 8 Web API
- React + Vite
- Tailwind CSS
- OpenAI GPT-4o-mini

## Getting Started

### Backend
```bash
dotnet restore
dotnet run
```

### Frontend
```bash
npm install
npm run dev
```
